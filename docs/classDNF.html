Detailed Description

Deep Neuronal Filter class.

It's designed to be as simple as possible with only a few parameters.

Parameters
nLayersNumber of layers
nTapsNumber of taps for the delay line feeding into the 1st layer

amThe activation function for the neurons. Default is tanh.
